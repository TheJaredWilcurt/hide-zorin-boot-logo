Change boot animation

To change the Zorin logo shown while booting up:

  1. Download this repo as a zip.
  2. Pull out the files in the "{{ os }}" folder and drop them into:
    • /usr/share/plymouth/themes/zorin-logo
    • Replacing the existing files.
  3. Run sudo update-initramfs -u to apply changes.
  4. Reboot to see the changes

How to copy files to a locked folder?

You'll need root permission. I installed a different file explorer called "Thunar". Then ran in the terminal sudo thunar to open it with root permissions. Then I navigated to the "{{ os }}" folder, select all, right-click > cut, then navigated to the /usr/share/plymouth/themes/zorin-logo folder and pasted them, saying "Replace all". After that you can close Thunar. Then run sudo update-initramfs -u to apply the changes and reboot.

Reverting the changes

If you want the Zorin logo back, do the exact same steps, but use the files in the "original" folder instead.

Creating custom animations

You can also create any custom animations you want.

  • The images must be exactly 556x140px.
  • The logo needs exactly 37 frames, named animation-00.png - animation-36.png.
  • The throbber needs exactly 39 frames, named throbber-00.png - throbber-38.png.

If you've created something you think others might also like, it can be added to this repo:

  • Images must be losslessly compressed. Use FileOptimizer with optimization level set to 9: Best. This is a Windows program, but installs on Zorin without any issues.
  • Create a pull request against the repo, or create an issue with the images attached in a zip.
  • If your images are not compressed correctly in your PR, I'll download them, compress them and create a new PR, closing yours out.
  • I'll also update the webpage to add the images.

Only tested on Zorin 18.
